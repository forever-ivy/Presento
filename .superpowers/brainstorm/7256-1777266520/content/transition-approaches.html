外层流程图进入详情页：3 个动画方向

推荐选 A：用现有 GSAP + React Flow camera，把“点击节点 → 聚焦 → 房间展开”做成一个连续镜头。

A. 节点共享元素展开（推荐）

点击节点后，React Flow 镜头先对准节点，节点生成一层光晕/白色底板，详情空间从该节点位置扩展出来。丝滑、清楚、和现有 GSAP 架构最贴合。

B. 地图后退 + 详情面板滑入

地图像工作台一样压到后景，详情从右侧进入。实现稳，但“点击节点进入空间”的空间关系没有 A 强，更像普通抽屉。

C. 保守淡入淡出

页面之间做简单 crossfade 和轻微缩放。风险最低，但视觉提升有限，不太满足“丝滑且视觉效果佳”。

我的推荐

选 A。实现上继续使用项目已经装好的 GSAP / @gsap/react 控制总时间线，保留 React Flow 的 setCenter 镜头动画；Framer Motion 只负责节点自身 hover/active 微动。这样不会混用两个库抢同一层 transform。
